How to package the applications into the exe files using pyinstaller

1. First you will need to install pyinstaller

In the PyCharm terminal type the following command

pip install pyinstaller

Next after the pyinstaller is installed, identify the script file that has your interface


pyinstaller --windowed --onefile interface_main.py

In your case the main script migth be called differenly substitute interface_main.py with yiur name

Once build is complete in your directory you will get two folders

  • build
  • dist
  • your_spript_name.spec in this case it is called interface_main.spec

Open the spec file. The contents should be similar to the ones shown below:

The lines you will need to change are highligted: 

# -*- mode: python ; coding: utf-8 -*-

a = Analysis(
    ['interface_main.py'],
    pathex=[],
    binaries=[],
    
datas=[],

    hiddenimports=[],
    hookspath=[],
    hooksconfig={},
    runtime_hooks=[],
    excludes=[],
    noarchive=False,
    optimize=0,
)
pyz = PYZ(a.pure)

exe = EXE(
    pyz,
    a.scripts,
    a.binaries,
    a.datas,
    [],
    name='interface_main',
    debug=False,
    bootloader_ignore_signals=False,
    strip=False,
    upx=True,
    upx_exclude=[],
    runtime_tmpdir=None,
    console=False,

 

    disable_windowed_traceback=False,
    argv_emulation=False,
    target_arch=None,
    codesign_identity=None,
    entitlements_file=None,
)

Finally run the command

pyinstaller script_name.spec

In this example pyinstaller interface_main.spec

Once the command will be finished, go to the dist folder and collect packaged .exe file


Each time you want to repackage scripts again after making changes you will only need to run pyinstaller script_name.spec
